# Disease-Prediction


This Project is aimed at prediction Disease based on given set of symptoms.

**ML Model Used** - Random Forest

**Language** - Python

**API** - Flask API


**API Input** - 

{ 
"symptoms" :
   
    [ 
       "itching",  
       "skin_rash", 
       "nodal_skin_eruptions",  
       "dischromic__patches"
    ]
}

**API OutPut**-

{
    "disease": "fungal_infection"
}
