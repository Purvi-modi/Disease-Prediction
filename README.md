# Disease-Prediction


This Project is aimed at prediction Disease based on given set of symptoms.

**ML Model Used** - Random Forest

**Language** - Python

**API** - Flask API


**API Input** - 

_{ "symptoms" :
    [ "itching",  
      "skin_rash", 
      "nodal_skin_eruptions",  
      "dischromic__patches"]
}_

**API OutPut**-

_{"disease": "fungal_infection"}_
